Description
-----------
This module provides a zip or postal code field type for CCK.

Features:
---------
* Validation of zip or postal codes for : 
	+ France, 
	+ UK, 
	+ Netherland (Dutch), 
	+ Canada, 
	+ USA, 
	+ Australia  
	+ Denmark 
	+ Portugal 
* Formating of zip or postal codes 

Prerequisites
-------------
The zipcode.module requires the content.module to be installed.

Installation
------------
To install, copy the zipcode directory and all its contents to your modules
directory.

Configuration
-------------
To enable this module, visit administer -> modules, and enable zipcode.

Bugs/Features/Patches:
